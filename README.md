# TikTok Claims Classification: Data Inspection & Preparation

 📌 Project Overview
As part of the TikTok data team, I performed initial data inspection and preparation for a machine learning model aimed at classifying videos as either **claims** or **opinions**. [cite_start]This project helps TikTok prioritize user reports and reduce moderator backlog.

🛠️ Tech Stack
* **Language:** Python
* **Libraries:** Pandas, NumPy
* **Framework:** PACE (Plan, Analyze, Construct, Execute)

 📈 Key Insights
* **Data Health:** Identified and documented 298 null values in engagement metrics.
* **Feature Engineering:** Developed `likes_per_view` and `shares_per_view` metrics to identify viral signals.
* **Findings:** Discovered that videos marked as "claims" from banned authors generate significantly higher median engagement than opinions.
